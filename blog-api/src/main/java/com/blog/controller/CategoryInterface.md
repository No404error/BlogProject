## 1.分类列表
#### 接口url：/categorys
#### 请求方式：GET
#### 请求参数：无
#### 响应数据：List\<CategoryVo>


## 2.详细分类列表
#### 接口url：/categorys/detail
#### 请求方式：GET
#### 请求参数：无
#### 响应数据：List\<Category>

## 3.分类信息
#### 接口url：/categorys/detail/{id}
#### 请求方式：GET
#### 请求参数：
| 参数名称 | 参数类型 |  说明  |
|:----:|:----:|:----:|
|  id  | long | 分类id |
#### 响应数据：Category
